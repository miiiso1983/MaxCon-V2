إصلاح شامل للقائمة الجانبية

📋 أو استخدم الكود مباشرة:

  1. اذهب إلى لوحة تحكم المستأجر
  2. اضغط F12 → Console
  3. انسخ والصق الكود أدناه:
var script = document.createElement('script'); script.src = 'https://www.maxcon.app/universal-sidebar-fix.js'; document.head.appendChild(script);

4. اضغط Enter وانتظر 5 ثوانٍ

✨ مميزات الإصلاح الشامل:

  • يبحث في جميع روابط الصفحة
  • يصلح أي رابط يحتوي على #
  • يضيف قوائم فرعية تعمل
  • يعمل مع أي نوع قائمة جانبية
  • إذا لم يجد قائمة، ينشئ قائمة عائمة
