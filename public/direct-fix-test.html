📋 التعليمات البديلة:

  1. اذهب إلى لوحة تحكم المستأجر
  2. اضغط F12 لفتح Developer Tools
  3. اختر تبويب Console
  4. انسخ والصق الكود أدناه:
var script = document.createElement('script'); script.src = 'https://www.maxcon.app/force-fix-sidebar.js'; document.head.appendChild(script);
