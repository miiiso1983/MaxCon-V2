⚠️ ملاحظة مهمة للخادم المباشر:

مسح الكاش على الخادم المباشر يتطلب وصول إلى cPanel أو SSH

1. مسح الكاش عبر cPanel

• اذهب إلى cPanel → File Manager

• احذف مجلد bootstrap/cache/

• احذف ملفات storage/framework/cache/

• احذف ملفات storage/framework/views/

2. إعادة تشغيل PHP

• في cPanel → PHP Selector

• اختر إعادة تشغيل PHP

• أو اتصل بمزود الاستضافة

3. مسح كاش المتصفح

• اضغط Ctrl + F5 (Windows) أو Cmd + Shift + R (Mac)

• أو استخدم نافذة خاصة/متخفية

4. التأكد من تسجيل الدخول

• تأكد من تسجيل الدخول كـ مستأجر وليس كمدير عام

• الرابط يظهر فقط في لوحة تحكم المستأجر

💡 نصائح للخادم المباشر:

  • إذا عملت الروابط المباشرة ولكن لا تظهر في القائمة الجانبية، فالمشكلة في الكاش
  • تأكد من تسجيل الدخول كمستأجر وليس كمدير عام
  • جرب متصفح آخر أو نافذة خاصة
  • قد تحتاج إلى الانتظار بضع دقائق بعد مسح الكاش

✅ علامات النجاح:

  • ظهور الصفحة التفاعلية مع جميع المحتويات
