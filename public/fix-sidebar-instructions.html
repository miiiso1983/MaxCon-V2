🔧 إصلاح قسم الإعدادات في القائمة الجانبية

حل سريع لإظهار قسم الإعدادات في القائمة الجانبية

❌ المشكلة:

صفحات الإعدادات تعمل بشكل ممتاز عند الوصول إليها مباشرة، لكن قسم الإعدادات لا يظهر في القائمة الجانبية على الخادم المباشر.

✅ الحل السريع:

تم إنشاء سكريبت JavaScript يضيف قسم الإعدادات ديناميكياً إلى القائمة الجانبية بدون الحاجة لتحديث ملفات Laravel.

⚡ الحل الفوري - إضافة السكريبت:

انسخ هذا الكود وأضفه في أي مكان في الموقع (مثل footer أو header):

<script src="https://www.maxcon.app/fix-sidebar-settings.js"></script>

الطريقة 1: إضافة السكريبت عبر cPanel

1. اذهب إلى cPanel → File Manager

2. افتح ملف resources/views/layouts/tenant.blade.php

3. أضف السكريبت قبل إغلاق tag الـ </body>

4. احفظ الملف

الطريقة 2: اختبار السكريبت مباشرة

1. اذهب إلى لوحة تحكم المستأجر

2. افتح Developer Tools (F12)

3. اذهب إلى Console

4. انسخ والصق الكود أدناه واضغط Enter:

var script = document.createElement('script'); script.src = 'https://www.maxcon.app/fix-sidebar-settings.js'; document.head.appendChild(script);

الطريقة 3: تحديث الملفات على الخادم

إذا كان لديك وصول SSH أو FTP:

1. حمّل آخر إصدار من GitHub

2. ارفع ملف resources/views/layouts/tenant.blade.php المحدث

3. امسح الكاش إذا أمكن

اختبر الحلول:

اذهب إلى النظام اختبار صفحات الإعدادات عرض السكريبت

🎯 ما سيحدث بعد تطبيق الحل:

  • سيظهر قسم "الإعدادات" في القائمة الجانبية
  • ستجد 6 روابط فرعية تعمل جميعها
  • ستظهر رسالة نجاح تؤكد إضافة القسم
  • جميع الروابط ستفتح في نوافذ جديدة
  • التصميم سيكون متناسق مع باقي القائمة
