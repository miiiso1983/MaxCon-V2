اختبار أهداف البيع

إصلاح الخطأ

الخطأ المكتشف:

Call to undefined method App\Http\Controllers\Tenant\SalesTargetController::middleware()

تم الإصلاح:

تم إزالة استدعاء middleware من constructor في SalesTargetController. Routes محمية بالفعل في ملف web.php.

الخطوات المطلوبة الآن:

1. تشغيل Migrations مطلوب
2. إضافة البيانات التجريبية مطلوب
3. تنظيف Cache مطلوب

الأوامر المطلوبة:

# تشغيل Migrations php artisan migrate # إضافة البيانات التجريبية php artisan db:seed --class=SalesTargetsSeeder # تنظيف Cache php artisan config:clear php artisan route:clear php artisan view:clear

روابط الاختبار

قائمة الأهداف إنشاء هدف جديد لوحة التحكم التقارير تسجيل الدخول

تشغيل الإعداد التلقائي

يمكنك تشغيل ملف الإعداد التلقائي لإكمال جميع الخطوات:

# في مجلد Laravel الرئيسي chmod +x setup_sales_targets.sh ./setup_sales_targets.sh

أو يمكنك تشغيل الأوامر يدوياً واحداً تلو الآخر.
