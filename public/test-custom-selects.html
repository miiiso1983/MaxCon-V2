القوائم المنسدلة المخصصة

نظام قوائم منسدلة موحد وقابل للبحث لجميع أجزاء النظام

قائمة أساسية

قائمة منسدلة أساسية مع إمكانية البحث
<select data-custom-select data-placeholder="--- اختر الدولة ---"> <option value="iraq">العراق</option> <option value="saudi">السعودية</option> </select>

اختيار متعدد

قائمة تسمح باختيار عدة عناصر مع خانات اختيار
<select data-custom-select multiple data-placeholder="اختر المهارات..."> <option value="php">PHP</option> <option value="javascript">JavaScript</option> </select>

خيارات محددة مسبقاً

قائمة مع خيارات محددة بشكل افتراضي
<select data-custom-select multiple> <option value="sales" selected>المبيعات</option> <option value="accounting" selected>المحاسبة</option> </select>

بيانات كبيرة

قائمة مع عدد كبير من الخيارات وبحث فعال
<select data-custom-select data-placeholder="ابحث عن المدينة..."> <!-- يتم تحميل الخيارات ديناميكياً --> </select>

بدون بحث

قائمة منسدلة بدون إمكانية البحث
<select data-custom-select data-searchable="false"> <option value="admin">مدير النظام</option> <option value="manager">مدير</option> </select>

تصميم مخصص

قائمة مع تصميم مخصص وحالات مختلفة
<select class="custom-select success" data-custom-select> <select class="custom-select error" data-custom-select>

كيفية التطبيق

  1. أضف ملفات CSS و JavaScript إلى صفحتك
  2. أضف الخاصية data-custom-select لأي عنصر select
  3. استخدم الخصائص الإضافية للتخصيص (data-placeholder, data-searchable, etc.)
  4. سيتم تحويل جميع القوائم تلقائياً عند تحميل الصفحة
  5. يمكن استخدام initCustomSelect(element) للتحويل اليدوي
